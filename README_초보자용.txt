노터치센트럴 Day 1 프로젝트

1. ZIP 압축을 풉니다.
2. 안의 모든 파일을 기존 바탕화면 NoTouchCentral 폴더 안으로 복사합니다.
3. VS Code에서 NoTouchCentral 폴더를 엽니다.
4. Terminal → New Terminal을 누릅니다.
5. 아래 두 줄을 차례로 입력합니다.

npm install
npm run dev

6. 터미널에 뜨는 http://localhost:5173 주소를 Ctrl 누른 채 클릭합니다.

현재 포함:
- 관리자 홈
- 블랙/화이트 테마 전환
- 관리자 실시간 지도 예시
- 이재인 지사장: BOSS 아이콘
- 나머지 관리자: TEAM LEADER 아이콘
- 모바일 하단 메뉴

현재 지도는 UI 확인용입니다. 실제 GPS/카카오 지도 연결은 다음 단계입니다.
